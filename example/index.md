# Welcome to mdstack

Drop a folder of markdown files. Get a rendered site. Each `.md` is a page; each frontmatter `title` is its nav label.

## How it works

- Every file in the folder becomes a route
- `index.md` is the homepage
- `order:` in frontmatter controls nav sort

> The design is intentionally plain right now. We'll dial it in next.

Take a look at the [Getting started](/getting-started/) page.
